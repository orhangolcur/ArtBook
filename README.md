# ArtBook

ArtBook is an Android application that allows users to save their favorite artworks. Users can add, edit, and view their saved artworks in a user-friendly interface. The project is developed using **Kotlin** and follows modern Android development practices.

## Features

- **Add Artwork**: Users can save artworks by entering their name, artist, and description.
- **View Artworks**: Saved artworks are displayed in an organized list.
- **Edit and Delete**: Users can update or remove artworks as needed.
- **Data Persistence**: Artworks are stored persistently and remain available even after closing the app.

## Technologies

- **Language**: Kotlin
- **Architecture**: MVVM (Model-View-ViewModel)
- **Database**: Room Database
- **UI**: Jetpack Compose or XML-based layouts (depending on the project implementation)
- **Other Libraries**: Glide or Picasso (for image handling), LiveData and ViewModel (for reactive programming)

## Requirements

- Android Studio Artic Fox or newer
- Minimum API Level: 21 (Android 5.0)
- Recommended API Level: 33 (Android 13)
